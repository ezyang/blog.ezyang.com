# Setting the page size

![image](xournal-slides/page-size.png)

The first thing to do is change the size of your pages into a more slide-like format. While you might be tempted to set your size to the standard size of 10in by 7.5in, I've found that it's a lot more convenient if the entire canvas fits on my screen, even with Xournal's toolbars visible. 8in by 6in, which has the same aspect ratio, fits nicely, and I keep a small Xournal file `blank-slides.xoj` as a template for my projects.
